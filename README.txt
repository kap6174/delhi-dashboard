# Urban Green Space & Park Accessibility Analysis - Team 07
## Delhi NCR Study

### Files Included:

**Core Deliverables:**
- Team_07_CleanedData.csv: Main cleaned dataset with all attributes
- Team_07_GreenSpaces.geojson: Complete spatial dataset (mixed geometries)
- Team_07_Delhi_Boundary.*: Study area boundary files

**Spatial Data (Shapefiles):**
- Team_07_GreenSpaces_Points.*: Point features only
- Team_07_GreenSpaces_Polygons.*: Polygon features only
- Team_07_GreenSpaces_All_Centroids.*: All features as centroids

**Analysis Results:**
- Team_07_Category_Statistics.csv: Green space categories analysis
- Team_07_Access_Statistics.csv: Access type breakdown
- Team_07_Size_Statistics.csv: Size distribution analysis

**Technical Files:**
- Team_07_Metadata.json: Complete processing metadata
- Team_07_GreenSpace_Density.tif: Density raster (if available)

### Data Sources:
- Primary: OpenStreetMap via Overpass API
- Boundary: OSM Nominatim
- Processing: Python/GeoPandas

### Coordinate System:
- EPSG:4326 (WGS84)
- Area calculations in UTM 43N
